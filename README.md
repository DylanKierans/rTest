# rTest - LibOTF2 Wrapper for R

This is a test package for creating a high-level otf2 wrapper for R. Wrapper built using [`Rcpp`](https://cran.r-project.org/web/packages/Rcpp/index.html).

## Dependencies

* LibOTF2
